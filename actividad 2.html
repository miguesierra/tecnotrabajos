Excel
Excel es una aplicacion destinada al manejo de bases de datos y la realizacion de operaciones para manejar esas bases de datos, usado condicionales, contadores y diferentes comandos hoy en clase trabajamos el comando SI, el comando Contar y Contar si, el comando promedio, suma y por ultimo el aleatorio, con este se realizo una pequeña base de datos para evaluar y tenre un rgistro de las perdidas y las ganadas de nuestra base de datos, el comando suma nos permite sumar, el comando promedio saca el promedio, el comando Si es para aplicar condiciones, el contar si es para contar valores si se cumple un valor determinado y el contar es para contar valores en general y por ultmio el comando aleatorio nos permite generar un numero aleatorio en un rango de numeros.
